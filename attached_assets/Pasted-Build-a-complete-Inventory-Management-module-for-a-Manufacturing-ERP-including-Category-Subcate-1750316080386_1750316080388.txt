Build a complete Inventory Management module for a Manufacturing ERP, including:

Category/Subcategory management
Add/Edit/Delete/View inventory items
Type tags: Product, Material, Spares, Assemblies
Customer Category-wise Pricing
Filter/search/sort
Role-based permission control (Add/Edit/Delete/View per module)

1. Schemas
Item
{
  name: String,
  code: String,
  category: String,
  subCategory: String,
  batch: String,
  qty: Number,
  unit: String,
  store: String,
  importance: String,
  type: String, // 'Product', 'Material', 'Spares', 'Assemblies'
  stdCost: Number,
  purchaseCost: Number,
  salePrice: Number,
  hsn: String,
  gst: Number,
  mrp: Number,
  internalManufacturing: Boolean,
  purchase: Boolean,
  description: String,
  internalNotes: String,
  minStock: Number,
  leadTime: Number,
  tags: [String],
  customerPrices: [
    { category: String, price: Number }
  ],
  createdAt: Date,
  updatedAt: Date
}
Category
{
  name: String,
  subCategories: [String]
}
CustomerCategory

{
  name: String,
  description: String
}
User
{
  username: String,
  email: String,
  password: String,
  role: String, // SuperUser, UnitHead, InventoryManager, etc.
  unit: String,
  permissions: {
    Inventory: {
      view: Boolean,
      add: Boolean,
      edit: Boolean,
      delete: Boolean
    }
  }
}
